# Product Overview

## Project Purpose
jobsearch-mcp-aws is an MCP (Model Context Protocol) server designed to integrate with popular UK job sites, providing programmatic access to job search functionality through AWS infrastructure.

## Key Features
- MCP server implementation for job search operations
- AWS cloud deployment using CloudFormation
- Python-based architecture with modern dependencies
- Support for UK job market integration
- Automated deployment scripts for development environments

## Target Users
- Developers building job search applications
- AI assistants requiring job market data access
- Organizations needing automated job posting retrieval
- Systems integrating with UK employment platforms

## Value Proposition
Provides a standardized MCP interface for accessing UK job sites, enabling seamless integration with AI tools and applications while leveraging AWS cloud infrastructure for scalability and reliability.

## Use Cases
- AI-powered job recommendation systems
- Automated job market analysis
- Career guidance applications
- Employment data aggregation services
- Integration with existing HR and recruitment tools
